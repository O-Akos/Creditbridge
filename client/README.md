# Szakdolgozat
Webalkalmazás az MSc-felvételi kreditátvitel adminisztratív támogatására.

Ez a webalkalmazás a BSc-t végzett hallgatók MSc képzésre történő jelentkezésének adminisztrációját támogatja.
A rendszer célja, hogy vizuálisan és interaktív módon segítse a kreditátviteli folyamatot az érintett hallgatók és az adminisztrátorok számára.
Az alkalmazás modern technológiákra épül, React alapú frontenddel és Node.js backenddel, PostgreSQL adatbázissal.

Technológiák:
Frontend: React
Backend: Node.js (Express)
Adatbázis: PostgreSQL
Verziókezelés: Git + GitHub

Fő funkciók:
Dinamikusan létrehozott szűrő
Adatbázisból olvasott tárgy lista
Kredittábla
Tantárgy elfogadó és előíró rendszer
Összesító lista az elfogadott és előírt tantárgyakból
Navigációs oldal a szakok között
Alap infromációs ablak megadása
